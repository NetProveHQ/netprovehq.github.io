# NetProve — Ultimate Windows Performance Optimizer

A free, lightweight Windows desktop optimization tool designed to maintain stable performance during long gaming sessions and video streaming.

## Features

- **Real-time Monitoring** — Live CPU, RAM, and network gauges with historical charts
- **Gaming Optimization** — Automatic gaming mode detection and process priority boosting
- **AI Assistant** — Built-in AI chatbot with personalized optimization recommendations
- **Network Optimization** — DNS benchmarking, TCP tuning, and latency reduction
- **Auto Mode** — Silent background optimization without interruption
- **Performance Reports** — Detailed session analytics and historical trends

## System Requirements

- Windows 10 / Windows 11 (x64)
- .NET 8 Runtime
- Administrator privileges
